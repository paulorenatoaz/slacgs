# Data

This folder is for dataset READMEs and small helper scripts. Do NOT store large datasets in this repository.

In CoInfoSim's dataset-anchored mode (Phase 2), real datasets are used to estimate class-conditional parameters from standardized information channels and to draw balanced subsets from a training reservoir. Describe datasets here and provide pointers to external sources or instructions to download and preprocess data for experiments. Dataset pipelines are planned and not yet implemented.
